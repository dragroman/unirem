# Безопасность данных

## Наш подход к безопасности данных

ООО «Универсал» придает первостепенное значение безопасности ваших персональных данных. Мы применяем современные технические, организационные и административные меры для защиты ваших персональных данных от несанкционированного доступа, использования, изменения, раскрытия или уничтожения.

## Технические меры защиты

Для обеспечения безопасности обработки и хранения ваших персональных данных мы применяем следующие технические меры:

### 1. Шифрование данных

- Использование протокола SSL/TLS для защиты передачи данных между вашим устройством и нашими серверами
- Шифрование конфиденциальной информации при хранении
- Применение надежных алгоритмов шифрования для защиты особо чувствительных данных, таких как платежная информация

### 2. Контроль доступа

- Строгая аутентификация для доступа к системам, содержащим персональные данные
- Многофакторная аутентификация для доступа к критическим системам
- Система управления правами доступа, основанная на принципе минимальных привилегий
- Регулярный аудит и пересмотр прав доступа

### 3. Защита инфраструктуры

- Использование брандмауэров, систем обнаружения и предотвращения вторжений
- Регулярное обновление программного обеспечения и систем безопасности
- Проведение сканирования уязвимостей и тестирования на проникновение
- Сегментация сети для ограничения распространения потенциальных угроз

### 4. Резервное копирование и восстановление

- Регулярное создание резервных копий данных
- Разработка и тестирование планов восстановления данных после сбоев
- Хранение резервных копий в зашифрованном виде в безопасных местах

## Организационные меры защиты

В дополнение к техническим мерам, мы реализуем следующие организационные меры для обеспечения безопасности ваших данных:

### 1. Политики и процедуры

- Разработка и внедрение всесторонних политик информационной безопасности
- Установление четких процедур обработки персональных данных
- Регулярное обновление политик и процедур в соответствии с изменениями в законодательстве и технологиях

### 2. Обучение сотрудников

- Проведение регулярного обучения сотрудников по вопросам безопасности и защиты данных
- Ознакомление сотрудников с обязанностями по защите персональных данных
- Обучение распознаванию и реагированию на инциденты безопасности

### 3. Управление инцидентами

- Разработка и внедрение процедур реагирования на инциденты безопасности
- Создание специальной команды по реагированию на инциденты
- Проведение регулярных тренировок по реагированию на инциденты
- Документирование и анализ инцидентов для предотвращения повторных нарушений

### 4. Оценка рисков

- Регулярное проведение оценки рисков безопасности
- Разработка мер по снижению выявленных рисков
- Мониторинг и переоценка рисков при изменении бизнес-процессов или технологической среды

## Ограничения безопасности

Несмотря на все наши усилия по обеспечению безопасности ваших данных, важно понимать, что ни один метод передачи через Интернет или метод электронного хранения не является 100% безопасным. Хотя мы стремимся использовать коммерчески приемлемые средства для защиты ваших персональных данных, мы не можем гарантировать их абсолютную безопасность.

## Уведомление о нарушениях безопасности

В случае нарушения безопасности, которое может затронуть ваши персональные данные, мы обязуемся:

1. Оперативно расследовать инцидент и принять необходимые меры для минимизации потенциального ущерба
2. Уведомить соответствующие регулирующие органы в соответствии с требованиями законодательства
3. Уведомить вас, если есть вероятность, что нарушение приведет к высокому риску для ваших прав и свобод
4. Предоставить вам информацию о мерах, которые вы можете принять для защиты от возможных негативных последствий нарушения

## Ваша роль в обеспечении безопасности

Безопасность ваших данных также зависит от вас. Мы рекомендуем следующие меры для обеспечения безопасности вашей информации:

- Используйте сложные пароли и не используйте один и тот же пароль для разных сервисов
- Не делитесь своими учетными данными с третьими лицами
- Убедитесь, что ваше устройство защищено антивирусным программным обеспечением
- Будьте внимательны к попыткам фишинга и социальной инженерии
- Регулярно проверяйте свои учетные записи на наличие подозрительной активности
- Выходите из системы после использования наших услуг, особенно при использовании общедоступных компьютеров

## Международные стандарты безопасности

Наши меры безопасности соответствуют российским и международным стандартам в области защиты информации, включая:

- ISO/IEC 27001:2013 (Системы управления информационной безопасностью)
- PCI DSS (для защиты платежной информации)
- Требования Федерального закона № 152-ФЗ «О персональных данных»
