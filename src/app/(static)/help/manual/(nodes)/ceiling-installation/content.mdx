# Инструкция по установке потолков

## 1. Инфракрасный уровень — инструмент №1

«Точность разметки — это 50% успеха работы. Доверяйте профессиональным инструментам!»

![Калибровка при помощи инфракрасного уровня](/images/help/Technology_1_1.jpg)

Калибровка при помощи инфракрасного уровня (разметка, проведение работ)

Для точной установки подвесного потолка необходимо использовать инфракрасный уровень. Этот инструмент гарантирует безупречную горизонтальную калибровку, что особенно важно перед разметкой и монтажными работами.

## 2. Соединение потолочных балок: шаг за шагом

Потолочные балки должны быть соединены как поперечно, так и продольно. Используйте только качественные крепежные элементы, такие как гвозди Jia Mei, чтобы гарантировать долговечность конструкции.

### Поперечное соединение

### Продольное соединение

![Продольное соединение балок](/images/help/Technology_1_3.jpg)

## 3. Технология гипсокартонных потолков

Работа с гипсокартоном требует внимательного подхода. Следуйте этим рекомендациям:

### Крепежные винты

Используйте винты длиной 15–18 см. Расстояние от края панели — не менее 1 см и не более 5 см.

![Крепление на винты](/images/help/Technology_1_4.jpg)

### Обработка стыков

Все стыки гипсокартона обрабатываются под прямым углом. V-образные швы оставляются для компенсации.

**Осторожно!** Стыки запрещено соединять по потолочной балке

## 4. Перегородки для потолочных балок

Установка перегородок производится на основании разметки. В случаях, когда на полу проходят трубопроводы, используйте клей, чтобы обеспечить крепление перегородки без повреждения труб.

- Разметка определяет точное положение перегородок.
- Крепежные точки располагаются в шахматном порядке.
- Перегородки не должны пересекаться с линиями сильного тока.
