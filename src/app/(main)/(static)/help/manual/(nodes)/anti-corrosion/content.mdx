# Антикоррозийная обработка винтов

Винты должны пройти антикоррозийную обработку. Для этого в цемент добавляют клей ПВА.

## 3. Нанесение гипса и установка угловых накладок

Нанесите гипс по разметке строительной линии внутренних и внешних углов.

На внешние углы установите угловые накладки по отвесу для обеспечения ровности. Таким образом, угол будет значительно сложнее ударить.

![Нанесение гипса 2](/images/help/Technology_2_5.jpg)

![Нанесение гипса 3](/images/help/Technology_2_6.jpg)

## 4. Выравнивание гипсовых участков

Для выравнивания областей с гипсом требуется освещение. Линии плинтусов выравниваются штукатуркой и шпатлевкой на уровне 50 мм от пола.

![Выравнивание гипсовых участков](/images/help/Technology_2_7.jpg)

Выравнивание гипсовых участков

Установка под отвесу для штукатурки плинтуса

## 5. Нанесение шпатлевки

Шпатлевку необходимо наносить 2-3 раза, согласно указанным рекомендациям.

## 6. Процесс шлифовки шпатлевкой

Процесс шлифовки шпатлевкой требует, чтобы все выходные коробки были обработаны ровно, а внутренняя часть очищена от мусора и грязи.

![Шлифовка шпатлевкой 1](/images/help/Technology_2_9.jpg)

![Шлифовка шпатлевкой 2](/images/help/Technology_2_10.jpg)

Все выходные коробки должны быть обработаны ровно

## 7. Завершающая шлифовка: контроль качества и очистка

Отшлифуйте шпатлевку, используя лампу, после шлифовки стены нужно очистить.

![Завершающая шлифовка](/images/help/Technology_2_11.jpg)

Шлифовка с помощью лампы

## 8. Нанесение эмульсионной краски

Нанесите эмульсионную краску с помощью малярного валика в 2-3 слоя, после этого уберитесь в ванной.
